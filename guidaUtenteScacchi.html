Download

Segui la guida per scaricare correttamente il file eseguibile
Per poter utilizzare il nostro applicativo devi aver già installato il pacchetto Java sul tuo computer, se non lo hai già installato CLICCA QUI

Istruzioni per l'Utilizzo

Questa sezione fornisce una guida dettagliata su come avviare il programma. Seguire i passaggi seguenti:

  1. Scarica il programma:
    Potrai cliccare su questo pulsante e scaricare il programma
  2. Aprire Sezione Download:
    Una volta cliccato Vai sulla sezione Download del tuo Browser e clicca sulla cartella per visuallizzare il file Scaricato
  3. Crea una cartella nel tuo disco C:
    Crea una cartella nel tuo disco C: con il nome "ProgettoScacchi" e poi inseriamo il nostro file Jar scaricato in precedenza

  4. Eseguiamo il nostro programma
    Andiamo sulla nostra barra di ricerca e scriviamo "PROMT DEI COMANDI" e eseguiamo

  5. Comando per eseguire il programma
    Adesso andiamo ad eseguire i comandi come da immagine e usiamo il comando "java -jar GestionaleScacchi.jar" per poter eseguire il nostro programma

Suggerimenti Utili:

  • Assicurarsi di aver scaricato correttamente il file Java sul tuo computer.
  • Assicurarsi di seguire correttamente tutti i passaggi qui sopra elencati.
  • Assicurarsi di seguire attentamente i passaggi dal punto 5, e eseguire correttamente i comandi elencati.
